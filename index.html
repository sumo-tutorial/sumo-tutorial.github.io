Erstelle deine eigene Webseite in wenigen Minuten!

Ein Projekt für den Kurs "Studier- und Medienkompetenz Online"

Einen GitHub Account anlegen

Um eine Webseite mit GitHub Pages anzulegen benötigt man zunächst einen Account bei GitHub. Diesen könnt ihr euch ganz einfach und kostenlos unter github.io anlegen. Wählt euren Benutzernamen sorgfältig, weil eure Webseite später unter der Adresse euer-benutzername.github.io erreichbar sein wird.

Damit GitHub Pages funktioniert, müsst ihr eure angegebene E-Mail Adresse bestätigen. Wenn ihr euch erfolgreich registriert habt, werdet ihr diesbezüglich eine E-Mail an die angegeben Adresse zugeschickt bekommen. Falls ihr nicht mehr sicher seid welche E-Mail Adresse ihr angegeben habt, diese ändern wollt oder die E-Mail nochmal zugeschickt bekommen wollt, könnt ihr dies in eueren Profil-Einstellungen bei GitHub unter "Emails" machen. Denkt dran, dass die E-Mail von GitHub auch im Spam-Ordner gelandet sein könnte.

GitHub Client installieren

Damit ihr später Daten zu GitHub hochladen könnt, müsst ihr den GitHub Client installieren. Das ist ebenfalls kostenlos und geht ziemlich fix. Das Programm kann unter den folgenden Links für Windows und Mac heruntergeladen werden.

Installation unter Windows: Wenn ihr die Installations-Datei fertig heruntergeladen habt, müsst ihr diese öffnen und einfach auf den "Installieren"-Button klicken. Den Rest wird dann von alleine erledigt - lediglich müsst ihr während der Installation noch eine Internet-Verbindung haben.

Sobald die Installation fertig ist, könnt ihr das Programm starten.

Ein neues Repository erstellen

Ein Repository ist eine Art Ordner, den GitHub für jedes Projekt erstellt. So ein Repository müssen wir jetzt erstellen, hierfür müsst ihr zunächst links oben auf Plus-Symbol klicken. Es öffnet sich ein Dialog der nach einem Namen und einem lokalen Pfad fragt. Als erstes müsst ihr den Namen von dem Repository eingeben. Dieser muss einer vorgebenen Konvention entsprechen damit GitHub weiß, dass genau dieses Repository geladen werden soll wenn man später auf eure Webseite geht.

Der Name des Repositorys wie folgt aussehen: euer-benutzername.github.io

Zum Beispiel heißt der Account für diese Webseite sumo-tutorial, dementsprechend muss das Repository sumo-tutorial.github.io genannt werden. Der lokale Pfad ist der Ort wo eine Kopie des Repositorys auf eurem Computer erstellt wird. Dieser Pfad ist für später wichtig, also merken! Aufgabe des GitHub Clients ist prinzipiell nicht mehr als dafür zu sorgen, dass der Inhalt des Ordners auf dem Server von GitHub und der Inhalt in dem Ordner auf eurem Computer gleich ist. Aber dazu später mehr.

Mit einem Klick auf "Create Repository" könnt die Erstellung abschließen. Wenn ihr jetzt den vorher angegeben lokalen Pfad öffnet werdet ihr sehen, dass GitHub dort einen Ordner mit dem angegeben Namen erstellt hat. In diesem Ordner sind zwei Dateien, die ihr ignorieren könnt.

Die Webseite erstellen

Die Vorbereitungen sind abgeschlossen! Öffnet den Ordner den GitHub für euer neues Repository unter dem angegeben lokalen Pfad erstellt hat. Hier könnt ihr jetzt eure HTML-Datei hinkopieren oder wenn ihr noch keine erstellt habt, hier erstellen. Keine Ahnung was eine HTML-Datei ist? Dann ladet einfach folgende Datei herunter und kopiert sie in den Ordner.
